## 📌 Sobre o projeto

Este repositório contém exercícios básicos em **Haskell**, feitos com foco em aprendizado de:

- Lógica de programação
- Funções recursivas
- Entrada de dados no terminal
- Condições (`if / then / else`)
- Conversão de tipos
- Sequências matemáticas

O objetivo é praticar programação funcional do zero.

---

## ⚙️ Requisitos

Antes de executar os códigos, você precisa ter instalado:

- GHC (Glasgow Haskell Compiler)
- GHCup (recomendado)

Colar no terminal antes de iniciar os codigos: 
- $env:Path += ";C:\ghcup\bin;C:\ghcup\ghc\9.6.7\bin"
  
Esse comando precisa ser executado sempre que abrir um novo terminal, a menos que você configure o PATH permanentemente no sistema.

Verifique se está instalado:
bash
(`ghc --version`)

## 🚀 Como executar os arquivos

Dentro da pasta do projeto, rode:
(`runghc nome_do_arquivo.hs`)
